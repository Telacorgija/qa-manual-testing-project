# qa-manual-testing-project
Manual QA project: test plan, test cases, bug reports (Jira).

This repository contains a manual QA testing project for a web application.

## Project Overview
The goal of this project is to demonstrate manual testing skills, including test design,
exploratory testing, and bug reporting.

## Scope
- Authentication (Registration, Login, Password Recovery)
- Catalog
- Community

## Test Artifacts
- Test Plan
- 60 Test Cases (created in Zephyr Scale)
- 21 Bug Reports (created in Jira)

## Testing Types
- Functional testing
- Exploratory testing
- Validation testing
- UI and basic UX testing

## Tools Used
- Jira
- Zephyr Scale
- Manual testing

## Test Environment
- Web application: https://demowebshop.tricentis.com/
- Browser: Microsoft Edge
- OS: Windows 11

## Notes
This project was created for learning and portfolio purposes.
